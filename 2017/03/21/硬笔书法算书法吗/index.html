硬笔书法算书法吗

发表于 2017-03-21 | 分类于 唠叨

上面说要在中小学推广书法学习。我没明白这里的“书法”是毛笔写的那种，还是钢笔、铅笔写的那种。现在的老人常批评年轻人写字越来越难看，怪他们总用电脑手机不爱动笔。其实更老的老人更早时也在批评，年轻人爱用钢笔，而不是毛笔和墨，所以写不好字。

我上小学时练习的是毛笔书法，那是学校里还有些年长的老师善用毛笔。每周有堂书法课，由他们来指导临帖。我记得主要是临“欧体”，可能因为“柳体”难学吧。我姥爷就是位小学教师，他写的魏碑体和“参考消息”上印的几乎一模一样。教室里常备一大瓶墨汁，上课时老师给每人的小碗里倒进一点。有时还要用它刷教室后面那块木黑板准备出“黑板报”。那是的墨汁好象很容易变臭，有时用它刷了黑板，满教室就得臭上三四天。

三十多年前有个叫庞中华的人，好象解决了“钢笔的问题”。他用钢笔写了本字帖，横竖撇捺模仿毛笔的效果，有顿笔有笔锋，整整齐齐，干干净净。我那时是初中生，我爸送了我一本。我的同学也几乎人手一本，老师也鼓励我们照着练习。

多少年过去了，如今小跑也上学了。他的老师也要求他练字，照着字帖练。字帖买回来了，我一看，还是庞中华的。据说如果算上巨量的盗版，这位庞先生的字帖可能是人类有史以来印刷量最高的印刷品，超了毛主席语录和圣经。

在庞先生的影响下我初中时字写得还不错。不过也有点小问题，班里很多人写一种字体，笔迹相互间高度近似，老师不象以前那样分得清谁写的。过了没多久，我不得不放弃那么写字了，只因为那样太慢。字写得再漂亮，考试时作文写不完老师也不给分。老师也改口了，说你们把字写清楚就行，别把自己当王羲之。谢谢老师的提醒，我又能正常地用钢笔写字了。我喜欢钢笔，到现在还在用。

后来我听说庞先生的本职是位地质队员，这下我就明白了。我毕业后也做地质工作，我的很多同学在各地的地质队里。地质队往往驻扎在偏僻之地，从前没有互联网，甚至有电视机但没电视节目信号。练书法是地质队里最常见的业余生活。要论书法高人，地质队里可是藏龙卧虎。只是他们整天灰头土脸，俯在吃饭打牌用的破桌子上，仿宋徽宗、米芾。而不能穿身唐装，挂条围巾，对着摄像机给大家写“宁静致远”“厚德载物”什么的。

小跑照着字帖写的这篇作业还不错，我想在老师那里可以通过。晚上电视新闻里，看到奥巴马左手笨拙地捏着笔，在斜放在桌面的纸上，写下七扭八歪的字母。我忽然想到会不会有人看到这一幕心中暗喜：美国这个国家快要完蛋了。
